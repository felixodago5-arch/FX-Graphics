# EduConnect Teacher Portal - Frontend

A complete frontend design for a Teacher Portal website where teachers can log in using Google Login and access learning materials.

## Features

- Clean, modern, professional design with academic theme
- Light and dark mode support
- Fully responsive design (mobile + desktop)
- Google Login integration (frontend simulation)
- Complete teacher portal with dashboard, materials, announcements, profile, and admin panel

## Pages Included

1. **Homepage** - Portal introduction with hero section and Google Login button
2. **Login Page** - Simplified Google Login interface
3. **Teacher Dashboard** - Overview with quick access cards and recent activity
4. **Teaching Materials Page** - File management with filtering and preview/download
5. **Announcements Page** - Announcement cards with titles, messages, and dates
6. **Profile Page** - Teacher details with edit functionality
7. **Admin Panel** - Upload materials, manage announcements, and subjects

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript (no frameworks)
- Font Awesome Icons
- Google Fonts (Inter)
- Flexbox/Grid for modern layouts

## Setup Instructions

1. Download or clone the project files
2. Open `index.html` in a web browser to view the homepage
3. Navigate between pages using the links in the header/sidebar
4. Test responsive design by resizing the browser window

## Key Features Implemented

### Theme System
- Light/dark mode toggle with localStorage persistence
- CSS variables for consistent theming across all pages

### Responsive Design
- Mobile-first approach with breakpoints at 576px, 768px, and 992px
- Collapsible sidebar navigation on mobile
- Adaptive layouts for all screen sizes

### Interactive Components
- Notification system for user feedback
- Filter functionality on materials page
- Simulated file upload and download
- Edit profile functionality (frontend only)
- Mobile menu with overlay

### UI/UX Elements
- Consistent card-based design
- Clear visual hierarchy
- Accessible color contrast
- Intuitive navigation
- Loading states and hover effects
